# LightSpeed ACHILLES Orchestration

This folder seeds the non-destructive orchestration layer for LightSpeed.

## Role

LightSpeed is the local/web interface shell, ACHILLES UI, dashboard, library, GMAT UI, adapter layer, and desktop orchestration surface.

## Rules

- Do not delete or overwrite current working widgets.
- Archive current raw widgets before modularisation.
- Treat LinkDrive as original evidence/source authority.
- Treat achilles.romer as final frontend/build orchestration space.
- Treat LightSpeed Desktop as the local Cognigrex spine, business/R&D digital twin, and pull/push orchestrator.
