# LightSpeed Repo Role

## Role

Local and web interface shell for ACHILLES, dashboard, library, GMAT UI, request console, and adapter contracts.

## Owns

- LightSpeed Desktop UI
- ACHILLES hub
- Dashboard
- Library UI
- GMAT UI
- Request console
- Desktop and web adapter contracts

## Does not own

- LinkDrive source authority
- Final MPL model authority
- Public website publication authority

## Feeds

- Sheets
- Romer-Protocol
- Data
- Operations

## Consumes

- LinkDrive index
- Sheets
- Romer-Protocol gateway
- Romer-MPL compact outputs
